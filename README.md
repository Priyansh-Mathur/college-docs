
# College Docs

A student-focused document sharing platform where students can upload and download study materials such as notes, PYQs (Previous Year Questions), and other academic files branch-wise / subject-wise / year-wise.

The goal of this project is to make academic resources easily accessible to all students in one place, so they can quickly find useful files from previous years and contribute their own materials for others.

---

## Features

- Upload academic documents and PYQs
- Download files shared by other students
- Access year-wise study materials
- Access subject-wise / paper-wise documents
- Centralized platform for student resources
- Simple structure for future expansion
- Ready to be deployed later

---

## Project Purpose

Students often struggle to find proper notes, previous year papers, and useful academic resources before exams. Many files remain scattered across WhatsApp groups, drives, and personal chats.

**College Docs** solves this problem by creating one common platform where students can:

- share useful files
- access documents uploaded by seniors or classmates
- download resources for any year or subject
- reduce dependency on random sources
